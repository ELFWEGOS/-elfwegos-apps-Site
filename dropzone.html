À quoi ça sert ?

DropZone est une application de bureau développée en Java qui permet d'organiser automatiquement les fichiers d'un dossier selon des règles personnalisables. L'objectif est simple : déplacer automatiquement les fichiers vers les bons emplacements.

Fonctionnalités actuelles

  • Création de règles de tri personnalisées
  • Déplacement automatique des fichiers selon leur extension
  • Gestion de plusieurs catégories
  • Sauvegarde de la configuration
  • Interface graphique moderne avec JavaFX
  • Journalisation (logs) des actions effectuées

Système de Mise à Jour Automatique

L'application gère ses propres mises à jour. Vous n'avez pas besoin de télécharger un nouveau dossier ou de remplacer vos fichiers manuellement, sauf cas très exceptionnel (si le système de mise à jour lui-même est modifié).

Note importante : Actuellement, lorsque vous lancez une mise à jour, l'application se ferme et rien ne s'affiche à l'écran pendant le téléchargement. Pas de panique ! L'application se rallumera toute seule une fois la mise à jour terminée. Une vraie fenêtre de progression de téléchargement sera ajoutée dans une prochaine version.

Installation & Lancement

L'application est 100% portable. Tout est déjà intégré à l'intérieur du dossier (Java, JavaFX, etc.). Vous n'avez aucun prérequis à installer sur votre ordinateur.

Pour lancer l'application, suivez ces deux étapes simples :

  1. Dézippez l'archive téléchargée où vous le souhaitez.
  2. Double-cliquez sur le fichier start.bat pour démarrer DropZone.
